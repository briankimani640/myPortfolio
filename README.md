
## Project Overview

This project is a multipage responsive portfolio website built using:
- HTML5 – for structure  
- CSS (Flexbox & Grid)– for layout and styling  
- JavaScript (Vanilla) – for interactivity and DOM manipulation  

It includes sections such as:
- Home: Introduction my small about section  
- About: Short bio with “Show More” functionality  
- Projects: A display of current and past works  
- Skills: My technical and soft skills  
- Contact: A form and links to reach me

## Features

- Responsive layout (works across devices)
- Smooth scrolling navigation
- Dynamic navbar color change on scroll
- “Show More” text toggle in About section
- Interactive button hover effects
- Section-based navigation links
- Clean and modern typography
